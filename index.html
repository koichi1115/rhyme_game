Dr.sibitt - Falling Rhymes Edition

A rhythm game focused on rhymes and rap skills

Download Game

Game Features

  • Falling words gameplay - catch the rhyming words!
  • Multiple difficulty levels based on combo count
  • Famous rap lyrics from well-known artists
  • Sound effects and background music
  • Progressive difficulty system

How to Play

Download the game, extract the files, and run falling_game.py with Python.

Requirements: Python 3.x and Pygame library

Installation

# Clone the repository
git clone https://github.com/koichi1115/rhyme_game.git

# Navigate to the game directory
cd rhyme_game

# Install pygame if you don't have it
pip install pygame

# Run the game
python falling_game.py
